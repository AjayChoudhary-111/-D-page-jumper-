🔥 What is Page Jumper?

Page Jumper is a lightweight Chrome extension that lets you control website pagination entirely from your keyboard.

No clicks.
No scrolling.
Just jump.

🎯 Features
. →   (>)→ Next page
, →   (>) Previous page
? →     Open jump-to-page input
Press ? again → Close input

Enter no. → Jump instantly

🔢 Badge shows current page number
🧭 Works per tab
🧼 Zero UI clutter
⚡ Ultra lightweight (Manifest V3)

🌍 Supported URL Patterns

/page/10/
?page=10
&page=10

Supports most common pagination systems for blogs, news, and listings.
